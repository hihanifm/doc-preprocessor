Endpoint and credentials are sent only with this request — not stored on the server. Use HTTPS if not on localhost. If the app runs in Docker and Ollama is on your computer, set base URL to http://host.docker.internal:11434/v1 (not 127.0.0.1 — that points at the container).

Lines starting with # in this box are treated as comments and skipped. Uses Python re syntax.

By section: either split at Word Heading styles (# lines) or at regex matches — useful when titles carry ids like x_y_z. Hints help the model map those ids to test_id. PDFs often lack # lines — try regex or whole file.

The server may throttle LLM traffic via LLM_RPM in .env (default 3 per 60s fixed window): you get up to 3 calls, then a pause until that minute-long window finishes and the limit fully resets. Local Ollama: often LLM_RPM=0.
